# Neki 🌙

**Neki** (meaning *goodness* or *virtue*) is a comprehensive and beautifully designed Islamic companion app built with Flutter. It aims to provide Muslims with essential daily tools in a modern, elegant, and user-friendly interface.

## ✨ Features

- **📖 Holy Quran**: Read the Quran with a clean interface. Includes a Surah list and dedicated reader screen, utilizing the `quran` package for accurate text and metadata.
- **🎧 Audio Recitations**: Listen to beautiful recitations using the robust `just_audio` package.
- **📚 Hadith Collection**: Browse and read through authenticated Hadiths.
- **🤲 Daily Duas**: A curated collection of supplications for various occasions.
- **📿 Digital Tasbih**: A convenient digital counter for your daily Dhikr.
- **🕌 Prayer Times & Location**: Built-in support for location-based features (powered by `geolocator`).
- **🌐 Community**: Connect with others, backed by Supabase.
- **🎨 Beautiful UI/UX**: Features a modern "Bento box" style layout, animated gradient backgrounds, and custom painters for Islamic motifs (crescent, mosque, stars).

## 🛠️ Tech Stack

This project leverages modern Flutter development practices and packages:

- **Framework**: [Flutter](https://flutter.dev/)
- **State Management**: [Riverpod](https://riverpod.dev/) (`flutter_riverpod`)
- **Backend as a Service**: [Supabase](https://supabase.com/) (`supabase_flutter`)
- **Audio Playback**: `just_audio`
- **Location**: `geolocator`
- **Typography**: `google_fonts`
- **Local Storage**: `shared_preferences`

## 🚀 Getting Started

### Prerequisites

- Flutter SDK (version `^3.11.1` or higher)
- Dart SDK
- iOS Simulator / Android Emulator or a physical device

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/SaminYeaserKhan/NEki.git
   cd NEki
   ```

2. **Install dependencies:**
   ```bash
   flutter pub get
   ```

3. **Run the app:**
   ```bash
   flutter run
   ```
